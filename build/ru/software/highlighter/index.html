Highlighter.pro

Это расширение для Chrome, написанное мной, которое работает как текстовый маркер на веб-страницах.

Я искал расширение с открытым кодом, которое можно установить как из исходного кода, так и из Chrome Web Store (кликните на эту ссылку для установки расширения), которое гарантированно не собирает никаких данных о пользователях, и, как это часто бывает, в результате напиcал свое.

Расширение очень быстрое, легковесное и не перегружает ваш браузер тяжелыми скриптами.

Функциональность​

  • Вы можете добавлять (и удалять) выделения на веб-странице с помощью контекстного меню браузера.
  • Расширение использует боковую панель для отображения списка выделений и заметок.
  • Вы можете добавлять или редактировать заметки к выделениям в боковой панели.
  • Заметки к выделениям можно писать в формате GitHub flavored Markdown. Это позволяет форматировать текст, добавлять заметки, списки задач, блоки кода и т.д.
  • Выделения и заметки можно сохранить в резервный файл, а затем восстановить на другом компьютере или в другом браузере Chrome. Для этого используйте страницу настроек расширения.
  • На странице настроек расширения вы можете удалить все выделения и заметки в текущем браузере (рекомендуется предварительно сохранить резервный файл).
  • Код открыт и опубликован на GitHub и GitLab.

Как сохранить и восстановить резервную копию​

Щелкните правой кнопкой мыши по значку расширения > выберите «Настройки» > используйте соответствующие кнопки для сохранения и восстановления резервной копии.

Поддержка​

Для получения поддержки напишите сообщение в репозитории GitHub.
