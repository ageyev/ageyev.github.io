# [Enlarge Arabic] — розширення для Chrome

Арабський текст на багатомовних веб-сторінках часто занадто дрібний, щоб його було зручно читати.
За однакового розміру шрифту арабські гліфи виглядають помітно дрібнішими за латинські, івритські або китайські символи —
це відома типографічна проблема, через яку перегляд статей у Вікіпедії, новинних сайтів, соціальних мереж і академічних сторінок із частиною текста арабскою стає надмірно ускладненим.

**[[Enlarge Arabic]](https://chromewebstore.google.com/detail/enlarge-arabic/cohiboahimednknkffilddeindcgpiop)** вирішує цю проблему одним кліком.

Розширення вибірково збільшує розмір арабського тексту на будь-якій веб-сторінці, не чіпаючи решту тексту.
Англійський, івритський, китайський чи будь-який інший текст, що знаходиться в тому ж абзаці, залишається в точності таким, як був — збільшується лише арабський.

![Порівняння: арабський текст на international-law.info до й після активації розширення — праворуч арабські гліфи помітно більші та зручніші для читання](/img/Illustrations/docs/tools/enlarge_arabic/before_and_after.png)

## Як користуватися

Натисніть на іконку розширення, щоб увімкнути збільшення арабського тексту на поточному сайті.
В активному стані іконка стає бірюзовою. Налаштування запам'ятовується для кожного сайту окремо — увімкніть його один раз, і воно діятиме під час кожного наступного відвідування, доки ви його не вимкнете.

### Налаштовувані параметри

Налаштувати розмір шрифту й міжрядковий інтервал для арабського тексту глобально можна на сторінці налаштувань ('Options') розширення:

![Сторінка налаштувань розширення з повзунками розміру шрифту (1.40em) та міжрядкового інтервалу (1.60), а також з попереднім переглядом у реальному часі](/img/Illustrations/docs/tools/enlarge_arabic/options.png)

Або для кожного сайту окремо (через бічну панель Chrome):

![Бічна панель Chrome з налаштуваннями для конкретного сайту на international-law.info, поточні значення: розмір шрифту 1.40em, міжрядковий інтервал 1.60](/img/Illustrations/docs/tools/enlarge_arabic/side_panel_open.png)


- Зміни відображаються миттєво завдяки попередньому перегляду в реальному часі
- Можна зберігати різні налаштування для різних сайтів — те, що підходить для Вікіпедії, може бути незручним для Twitter

## Працює всюди

- Обробляє динамічно завантажуваний контент (нескінченна прокрутка, стрічки в реальному часі, односторінкові застосунки)
- Зберігає зв'язне (курсивне) написання арабського — літери з'єднуються одна з одною природно, як і має бути
- Автоматично адаптується до світлої або темної теми браузера

## Поважає вашу приватність

Жодного збору даних. Жодної аналітики. Жодних мережевих запитів. Усі налаштування зберігаються локально на вашому пристрої і ніколи не залишають ваш браузер.
