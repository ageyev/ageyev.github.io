[Enlarge Arabic] — розширення для Chrome

Арабський текст на багатомовних веб-сторінках часто занадто дрібний, щоб його було зручно читати. За однакового розміру шрифту арабські гліфи виглядають помітно дрібнішими за латинські, івритські або китайські символи — це відома типографічна проблема, через яку перегляд статей у Вікіпедії, новинних сайтів, соціальних мереж і академічних сторінок із частиною текста арабскою стає надмірно ускладненим.

[Enlarge Arabic] вирішує цю проблему одним кліком.

Розширення вибірково збільшує розмір арабського тексту на будь-якій веб-сторінці, не чіпаючи решту тексту. Англійський, івритський, китайський чи будь-який інший текст, що знаходиться в тому ж абзаці, залишається в точності таким, як був — збільшується лише арабський.

Як користуватися​

Натисніть на іконку розширення, щоб увімкнути збільшення арабського тексту на поточному сайті. В активному стані іконка стає бірюзовою. Налаштування запам'ятовується для кожного сайту окремо — увімкніть його один раз, і воно діятиме під час кожного наступного відвідування, доки ви його не вимкнете.

Налаштовувані параметри​

Налаштувати розмір шрифту й міжрядковий інтервал для арабського тексту глобально можна на сторінці налаштувань ('Options') розширення:

Або для кожного сайту окремо (через бічну панель Chrome):

  • Зміни відображаються миттєво завдяки попередньому перегляду в реальному часі
  • Можна зберігати різні налаштування для різних сайтів — те, що підходить для Вікіпедії, може не підійти для X (Twitter), Facebook або вашого місцевого сайту новин.

Працює всюди​

  • Обробляє динамічно завантажуваний контент (нескінченна прокрутка, стрічки в реальному часі, односторінкові застосунки)
  • Зберігає зв'язне (курсивне) написання арабського — літери з'єднуються одна з одною природно, як і має бути
  • Автоматично адаптується до світлої або темної теми браузера

Поважає вашу приватність​

Жодного збору даних. Жодної аналітики. Жодних мережевих запитів. Усі налаштування зберігаються локально на вашому пристрої і ніколи не залишають ваш браузер.

Відкритий код. Повний код доступний на GitHub.

Про дозволи. Chrome показує, що це розширення запитує дозвіл «читати та змінювати дані на сайтах, які ви відвідуєте». Це необхідно, тому що розширення має знаходити арабський текст на будь-якій сторінці, яку ви відкриваєте, і регулювати його розмір. Розширення не збирає, не передає та не зберігає дані про сайти, які ви відвідуєте, — його єдине призначення полягає в тому, щоб зробити арабський текст зручнішим для читання. Повний код доступний на GitHub для перевірки.
