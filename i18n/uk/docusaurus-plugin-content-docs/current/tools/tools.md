# Інструменти та ресурси

У процесі моєї роботи в галузі міжнародного права — зокрема в його перетині з релігійним правом, особливо в контексті близькосхідного конфлікту — та в розробці юридичних і фінансових технологій
я регулярно створюю інструменти та довідкові матеріали для вирішення завдань, з якими стикаюся.

Деякі з них виявилися корисними поза межами моєї власної роботи, і я ділюся ними тут у надії, що вони стануть в пригоді й іншим.

Усе програмне забезпечення має відкритий код і доступне на [GitHub](https://github.com/ageyev).

## Інструменти для роботи з текстом

[**Highlighter.pro**](/tools/highlighter) — розширення для Chrome, що дозволяє виділяти текст на будь-якій веб-сторінці. Містить бічну панель для навігації між виділеннями та нотатки у форматі Markdown, прикріплені до кожного виділення.

## Арабська мова

[**[Enlarge Arabic]**](/tools/enlarge_arabic) — розширення для Chrome, що збільшує арабський текст на будь-якому сайті — зокрема всередині багатомовних сторінок — для зручності читання.

[**Арабська абетка з відповідниками в івриті**](pathname:///html/arabic_alphabet_with_forms.html) — таблиця-довідник з арабських літер для тих, хто вже знайомий з івритом. Кожна арабська літера зіставлена з івритським когнатом, із зауваженнями про спільні протосемітські корені, звукові відповідності та чотири позиційні форми (початкова, серединна, кінцева, ізольована).

[**Хамзату-ль-васль — довідник**](pathname:///html/hamzat_al_wasl_reference.html) — структурований довідник зі сполучної хамзи в арабській мові: означений артикль, десять канонічних імен, відносні займенники та дієслівні моделі. З повною огласовкою, транслітерацією, івритськими когнатами та коранічними прикладами.

## Криптографія та безпека

[**pgp-tools**](https://github.com/ageyev/pgp-tools) — бібліотека на Java, що спрощує основні операції OpenPGP (шифрування, підпис, керування ключами) на основі фреймворку BouncyCastle.

## Утиліти для розробників

[**api-test-server**](https://github.com/ageyev/api-test-server) — простий локальний сервер на Node.js для тестування HTTP-запитів. Записує вхідні GET/POST-дані в консоль і у файл журналу; корисний під час налагодження при розробці API.
