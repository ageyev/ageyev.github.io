# Инструменты и ресурсы

В процессе моей работы в области международного права — включая его пересечение с религиозным правом, особенно в контексте ближневосточного конфликта — и в разработке юридических и финансовых технологий
я регулярно создаю инструменты и справочные материалы для решения задач, с которыми сталкиваюсь.

Некоторые из них оказались полезными за пределами моей собственной работы, и я делюсь ими здесь в надежде, что они пригодятся и другим.

Всё программное обеспечение имеет открытый код и доступно на [GitHub](https://github.com/ageyev).

## Инструменты для работы с текстом

[**Highlighter.pro**](/tools/highlighter) — расширение для Chrome, позволяющее выделять текст на любой веб-странице. Включает боковую панель для навигации между выделениями и заметки в формате Markdown, прикреплённые к каждому выделению.

## Арабский язык

[**[Enlarge Arabic]**](/tools/enlarge_arabic) — расширение для Chrome, увеличивающее арабский текст на любом сайте — в том числе внутри многоязычных страниц — для удобства чтения.

[**Арабский алфавит с соответствиями в иврите**](/html/arabic_alphabet_with_forms.html) — таблица-справочник по арабским буквам для тех, кто уже знаком с ивритом. Каждая арабская буква сопоставлена с ивритским когнатом, с замечаниями об общих протосемитских корнях, звуковых соответствиях и четырёх позиционных формах (начальная, серединная, конечная, изолированная).

[**Хамзату-ль-васль — справочник**](/html/hamzat_al_wasl_reference.html) — структурированный справочник по соединительной хамзе в арабском языке: определённый артикль, десять канонических имён, относительные местоимения и глагольные модели. С полной огласовкой, транслитерацией, ивритскими когнатами и кораническими примерами.

## Криптография и безопасность

[**pgp-tools**](https://github.com/ageyev/pgp-tools) — библиотека на Java, упрощающая основные операции OpenPGP (шифрование, подпись, управление ключами) на основе фреймворка BouncyCastle.

## Утилиты для разработчиков

[**api-test-server**](https://github.com/ageyev/api-test-server) — простой локальный сервер на Node.js для тестирования HTTP-запросов. Записывает входящие GET/POST-данные в консоль и в файл журнала; полезен при отладке во время разработки API.
