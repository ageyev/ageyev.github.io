# [Enlarge Arabic] — расширение для Chrome

Арабский текст на многоязычных веб-страницах часто слишком мелкий, чтобы его было удобно читать.
При одном и том же размере шрифта арабские глифы выглядят заметно мельче латинских, ивритских или китайских символов —
это известная типографическая проблема, из-за которой просмотр статей в Википедии, новостных сайтов, социальных сетей и академических страниц с включением арабского текста становится излишне затруднительным.

**[[Enlarge Arabic]](https://chromewebstore.google.com/detail/enlarge-arabic/cohiboahimednknkffilddeindcgpiop)** решает эту проблему одним кликом.

Расширение избирательно увеличивает размер арабского текста на любой веб-странице, не затрагивая остальной текст.
Английский, иврит, китайский или любой другой текст, находящийся в том же абзаце, остаётся в точности как был — увеличивается только арабский.

![Сравнение: арабский текст на international-law.info до и после активации расширения — справа арабские глифы заметно крупнее и удобнее для чтения](/img/Illustrations/docs/tools/enlarge_arabic/before_and_after.png)

## Как пользоваться

Нажмите на иконку расширения, чтобы включить увеличение арабского текста на текущем сайте.
При активном состоянии иконка становится бирюзовой. Настройка запоминается для каждого сайта отдельно — включите её один раз, и она будет действовать при каждом следующем посещении, пока вы её не отключите.

### Настраиваемые параметры

Настроить размер шрифта и межстрочный интервал для арабского текста глобально можно на странице настроек ('Options') расширения:

![Страница настроек расширения с ползунками размера шрифта (1.40em) и межстрочного интервала (1.60), а также с предпросмотром в реальном времени](/img/Illustrations/docs/tools/enlarge_arabic/options.png)

Или для каждого сайта отдельно (через боковую панель Chrome):

![Боковая панель Chrome с настройками для конкретного сайта на international-law.info, текущие значения: размер шрифта 1.40em, межстрочный интервал 1.60](/img/Illustrations/docs/tools/enlarge_arabic/side_panel_open.png)


- Изменения отображаются мгновенно благодаря предпросмотру в реальном времени
- Можно сохранять разные настройки для разных сайтов — то, что подходит для Википедии, может быть не идеально для X (Твиттера), Facebook или вашего местного новостного сайта.

## Работает везде

- Обрабатывает динамически загружаемый контент (бесконечная прокрутка, ленты в реальном времени, одностраничные приложения)
- Сохраняет связное (курсивное) написание арабского — буквы соединяются друг с другом естественно, как и должны
- Автоматически адаптируется к светлой или тёмной теме браузера

## Уважает вашу приватность

Никакого сбора данных. Никакой аналитики. Никаких сетевых запросов. Все настройки хранятся локально на вашем устройстве и никогда не покидают ваш браузер.

**Открытый код**. Полный исходный код доступен на GitHub: https://github.com/ageyev/enlarge-arabic 

**О разрешениях**. Chrome показывает, что это расширение запрашивает разрешение «читать и изменять данные на посещаемых вами сайтах». Это необходимо потому, что расширение должно находить арабский текст на любой открываемой вами странице и регулировать его размер. Расширение не собирает, не передаёт и не хранит данные о посещаемых вами страницах — его единственное назначение состоит в том, чтобы сделать арабский текст более удобным для чтения. Полный исходный код доступен на GitHub для проверки.
