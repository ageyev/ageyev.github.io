Highlighter.pro

Це розширення для Chrome, написане мною, яке працює як текстовий маркер на вебсторінках.

Я шукав розширення з відкритим кодом, яке можна встановити як із вихідного коду, так і з Chrome Web Store (натисніть на посилання для інсталяції), яке гарантовано не збирає жодних даних про користувачів, та, як часто буває, в результаты написав своє.

Розширення дуже швидке, легке та не перевантажує ваш браузер важкими скриптами.

Ось як воно працює:

Функціональність​

  • Ви можете додавати (та видаляти) виділення на веб-сторінці за допомогою контекстного меню браузера.
  • Розширення використовує бічну панель для відображення списку виділень і нотаток.
  • Ви можете додавати або редагувати нотатки до виділень у бічній панелі.
  • Нотатки до виділень можна писати у форматі GitHub flavored Markdown. Це дозволяє форматувати текст, додавати нотатки, списки завдань, блоки коду тощо.
  • Виділення та нотатки можна зберегти у резервний файл, а потім відновити на іншому комп’ютері або в іншому браузері Chrome. Для цього використовуйте сторінку налаштувань розширення.
  • На сторінці налаштувань розширення ви можете видалити всі виділення та нотатки у поточному браузері (рекомендується попередньо зберегти резервний файл).
  • Код відкритий і опублікований на GitHub і GitLab.

Як зберегти та відновити резервну копію​

Клацніть правою кнопкою миші на значку розширення > виберіть «Налаштування» > використовуйте відповідні кнопки для збереження та відновлення резервної копії.

Підтримка​

Для отримання підтримки напишіть повідомлення в репозиторій GitHub.
